# GameVault Database

Bu proje, bir oyun platformu için PostgreSQL uyumlu bir veritabanı tasarımını içermektedir.

## İçerik
- Veritabanı tabloları (SQL)
- Primary Key ve Foreign Key ilişkileri
- ER Diyagramı (drawSQL)

## Tablolar
- developers
- games
- genres
- games_genres

## Kullanılan Teknolojiler
- PostgreSQL
- SQL
- drawSQL
